Glosario

  • MERN:
    MongoDB (Base de datos)
    Express (Servidor)
    ReactJS (Librería Frontend)
    NodeJS (Lenguaje de programación Backend)

  • HTML: HyperText Markup Language
    Es un lenguaje de marcado para la elaboración de páginas Web.

  • CSS: (Cascading Style Sheets)
    Es un lenguaje de diseño gráfico para definir y crear la presentación de un documento estructurado escrito en un lenguaje de marcado.

  • Javascript
    Javascript es un lenguaje de programación interpretado.
    Se usó inicialmente en el navegador (frontend).
    En la actualidad se usa tanto en frontend y en Backend.
    Todos los navegadores soportan de manera nativa.
    Javascript NO es JAVA.

  • Frontend
    Es el desarrollado del lado del cliente (navegador)
    Todo lo que se ve

  • Backend
    Es la parte del lado del servidor mediante el uso de lenguajes de programación.
    Es la lógica de como funciona la aplicación

  • Full Stack
    Quien tiene conocimiento de Frontend y Backend.
    Puede iniciar un proyecto desde cero y terminarlo.

  • Librería
    Es un conjunto de funcionalidades que resuelven necesidades específicas del proyecto.
    Ejemplo: ReactJS

  • Framework
    Te da toda la estructura para un proyecto completo, desde el inicio hasta el final integrando funcionalidades sin depender de librerías externas.
    Ejemplo: Ruby on Rails

  • Repositorio
    Es el lugar donde se guarda el código mediante un control de versiones (Git)

  • Terminal / Consola
    Es el lugar donde se ve y se ejecuta código.

  • IDE
    Es el lugar donde se escribe el código.

  • SEO
    (Search engine optimization) Es un conjunto de acciones orientadas a mejorar el posicionamiento de un sitio web en la lista de resultados.
